Git

  • Status - отображает неотслеживаемые файлы в удаленном репозитории, название текущей ветки, рекомендация по дальнейшим шагам и тд.
  • Commit -m "Название"- позволяет создать коммит/сохранить изменения.
  • Push - отправляет все изменения на удаленный репозиторий.
  • Pull - отправляет изменения из удаленного репозитория на локальный.
  • Branch Название ветки - создание новой ветки
  • Add - создание нового файла.
  • Log - список коммитов.
  • Checkout - переход между ветками.
  • Clone - клонирование репозитория из удаленного в локальный.
  • Help - список команд Git
