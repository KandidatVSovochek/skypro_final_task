Терминал MacOS

  • CD - перемещение между папками.
  • PWD/WIN - текущая рабочая директория
  • Is - список файлов текущей директории
  • mkdir - создать мамку в текущей директории
  • Rm название файла - удалить файл
  • Rm -r название файла - удалить папку
  • Clear - очистить терминал
